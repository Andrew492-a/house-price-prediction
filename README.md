# House Price Prediction

## Project Overview

This project focuses on predicting house prices using machine learning regression models.

The dataset contains multiple features describing residential homes, such as:

- Living area
- Number of rooms
- Garage size
- Year built
- Basement size

The goal is to build predictive models that estimate house prices based on these features.

## Dataset

House Prices Dataset from Kaggle.

https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques

## Project Workflow

1. Data Exploration
2. Data Cleaning
3. Handling Missing Values
4. Feature Engineering
5. Model Training
6. Model Evaluation

## Regression Models Used

- Linear Regression
- Ridge Regression
- Lasso Regression
- Random Forest Regressor
- Gradient Boosting Regressor
- Decision Tree Regressor

## Evaluation Metrics

- RMSE
- R² Score
- Mean Absolute Error

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Project Outcome

The models were evaluated to determine which algorithm provides the best prediction performance for housing prices.

This project demonstrates the application of regression models in real-world predictive analytics.
